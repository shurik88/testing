Проверяем: получение списка с прогнозом погоды

Предусловие - количество выводимых значений должно быть от 1 до 10. Значение по умолчанию: 5


1) вводим число от 1 до 10 (не 1 и не 10) в поле count, вызываем метод get
 1.1) Получаем статус код 200 - успешно выведено количество строк, соот-щее введенному числу от 1 до 10
схема:
date	string($date-time)
Дата
temperatureC	integer($int32)
Температура по Цельсию
temperatureF	integer($int32)
readOnly: true
Температура по Фаренгейту
summary	string
nullable: true

 1.2) Проверяем данные:
[

    "date": "2020-10-22T11:34:54.9382031+00:00",
    "temperatureC": 21,
    "temperatureF": 69,
    "summary": "Cool"
  }
]

2) Проверяем граничные значения. Вводим в count 1, вызываем метод Get
 2.1) Получаем статус код 200 - успешно выведена одна строка

3) Проверяем граничные значения. Вводим в count 10, вызываем метод Get
 3.1) Получаем статус код 200 - успешно выведено 10 строк

4) Вводим в count 0, вызываем метод Get , получаем статус код 400
тело ответа: Count value should be between 1 and 10
